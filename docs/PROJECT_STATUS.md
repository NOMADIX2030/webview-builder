## 1. 현재 상태

### 1.1 완료된 기능 (웹뷰 APK 빌드)

| 구분 | 내용 |
|------|------|
| **1단계** | 웹 URL, 앱 아이콘(512×512 권장), 스플래시 업로드 |
| **2단계** | 도메인 기반 자동 생성값(앱 이름, 패키지 ID 등) 확인·수정 |
| **3단계** | 모바일 시뮬레이션, APK 적용 아이콘 미리보기, 빌드 요청 |
| **빌드** | Capacitor 템플릿, Android APK, Keystore 자동 생성 (동기 실행, 2~5분) |
| **다운로드** | APK, Keystore 다운로드 |

### 1.2 알려진 이슈

| 이슈 | 설명 | 우선순위 |
|------|------|----------|
| 3단계 아이콘 미리보기 비율/여백 | 3단계에서 보이는 아이콘 비율·여백이 실제 APK와 다름 | 낮음 (추후 개선) |

# 네이티브 하단 탭바 기능 개발 예정안

> **상태**: 개발 예정 (미구현)  
> **작성일**: 2026-03-06  
> **목적**: 하단 네비게이션이 없는 웹사이트에 앱 레벨 네이티브 탭바를 선택적으로 추가하는 기능

---

## 1. 기획 배경

### 문제
- 일부 웹사이트는 하단 네비게이션이 없거나 웹 UI가 앱답지 않음
- 단순 웹뷰만으로는 네이티브 앱 경험 제공 불가

### 해결
- 빌드 시 **옵션으로 네이티브 하단 탭바 추가** 가능
- 웹에 하단 네비가 있는 경우 → 옵션 OFF (기존과 동일)
- 웹에 하단 네비가 없는 경우 → 옵션 ON (앱이 대신 제공)

### 핵심 컨셉
```
┌──────────────────┐
│                  │
│   WebView        │  ← 웹 콘텐츠
│   (탭 내용)      │
│                  │
├──────────────────┤
│  🏠   💬   🔔   👤  │  ← 네이티브 하단 탭바 (앱 레벨)
└──────────────────┘
```

## 2. 기능 명세

### 2.1 탭 기본 사양

| 항목 | 사양 |
|---|---|
| 탭 개수 | 최소 2개 ~ 최대 5개 |
| 탭 구성 | 아이콘 + 텍스트 + URL |
| 아이콘 방식 | SVG 코드 직접 입력 |
| 활성 탭 표시 | 색상 변경 (활성/비활성 구분) |
| 탭 클릭 동작 | WebView에 해당 URL `loadUrl` |

### 2.2 디자인 커스텀 항목

| 항목 | 설정 방법 | 예시 |
|---|---|---|
| 배경색 | HEX 색상 입력 | `#FFFFFF`, `#1A1A1A` |
| 활성 탭 색상 | HEX 색상 입력 | `#007AFF` (파란색) |
| 비활성 탭 색상 | HEX 색상 입력 | `#8E8E93` (회색) |
| 아이콘 | SVG 코드 입력 | Heroicons, Material Icons 등 |
| 탭 텍스트 | 자유 입력 | 홈, 채팅, 알림, 마이 |

### 2.3 아이콘 SVG 입력 방식

사용자가 외부 아이콘 사이트에서 SVG 코드를 복사하여 붙여넣기:

```
추천 아이콘 소스:
- Heroicons (heroicons.com) — MIT 라이선스
- Material Icons (fonts.google.com/icons) — Apache 2.0
- Feather Icons (feathericons.com) — MIT 라이선스
- Phosphor Icons (phosphoricons.com) — MIT 라이선스
```

입력 예시:
```xml
<svg xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24" stroke-width="1.5" stroke="currentColor">
  <path stroke-linecap="round" stroke-linejoin="round" d="m2.25 12 8.954-8.955c.44-.439 1.152-.439 1.591 0L21.75 12M4.5 9.75v10.125c0 .621.504 1.125 1.125 1.125H9.75v-4.875c0-.621.504-1.125 1.125-1.125h2.25c.621 0 1.125.504 1.125 1.125V21h4.125c.621 0 1.125-.504 1.125-1.125V9.75M8.25 21h8.25" />
</svg>
```

## 4. 기술 구현 계획

### 4.1 데이터 흐름

```
Step 2 폼 입력

→ Build 모델 config_json['bottom_nav'] 저장
→ CapacitorBuildService에서 탭바 코드 주입

```

## 9. 참고 사항

### SVG → VectorDrawable 변환 참고
- Android Studio 내장 변환 도구 존재
- [svg2vector](https://github.com/nicholasgasior/svg2vector) PHP 라이브러리
- 단순 아이콘 SVG는 변환 성공률 높음
- 복잡한 그라디언트/필터 SVG는 변환 실패 가능 → 입력 시 안내 필요

### 현재 탭 활성화 동기화
WebView `onPageFinished`에서 현재 URL을 파싱하여 해당 탭을 자동 활성화:
```java
// 예: /chat으로 시작하면 채팅 탭 활성
if (url.startsWith(baseUrl + "/chat")) {
    bottomNav.setSelectedItemId(R.id.tab_chat);
}
```
